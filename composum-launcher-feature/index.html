About Composum Launcher Feature Launchers

Composum launchers using the feature system

Project Modules

This project has declared the following modules:

Name Description
Composum Launcher Feature Nodes Starter Creates a feature set and a starter based on a Sling Starter 12 snapshot to start Sling and updating Composum Nodes. It produces a self contained feature archive that doesn't need network access to start. The starter contains that feature archive.
Composum Launcher Feature Composum Starter (With Composum Public) Creates a feature set and a starter based on a Sling Starter 12 snapshot to start Sling, updating Composum Nodes and deploying all public parts of composum. It produces a self contained feature archive that doesn't need network access to start. The starter contains that feature archive.
Composum Launcher Feature Felixcontainer Creates a zip containing the felix artifact needed to start a feature launcher
composum-launcher-feature-integrationtest Integration test for launchers, to be used with scope test.
